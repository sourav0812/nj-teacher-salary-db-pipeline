# NJ Teacher Salary Database Pipeline

## Overview
A Python-based data pipeline that connects to a MySQL database, 
creates a relational schema, and loads 40,000+ rows of New Jersey 
public school teacher salary data from a CSV file.

## Skills & Tools
- Python, MySQL, mysql-connector-python
- SQL (CREATE TABLE, LOAD DATA INFILE)
- Database design, ETL (Extract, Transform, Load)

## What This Project Does
1. Connects Python to a local MySQL server
2. Creates a new database and table schema
3. Loads real NJ state teacher salary data from a CSV file
4. Validates row and column counts after loading

## Data Source
New Jersey public teacher salary records (~40,000 rows, 16 columns)
